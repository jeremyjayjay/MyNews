## 项目简介
    具体功能： 
    
    1.挑战国内各大中文资讯站点新闻的数据挖掘，并按各板块新闻数据分类存储； 
    
    2.为了方便演示，只爬取娱乐，财经，体育，科技四大板块新闻；
    
    3.数据清洗并入mysql库 

    4.使用celery消息队列工具，自定义程序自动运行和间隔动态运行，实现数据的动态获取 
 
    5.使用redis保存celery中borker消息的队列缓存,实现异步任务调度实现定时存储 
    
    6.采用Redis做url动态缓存，实现分布式爬取系统 
## 环境要求:
    环境及模块:python3+scrapy-redis框架+Fiddler抓包工具+celery（要求版本4.1.0）+ redis +mysql
